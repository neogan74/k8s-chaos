{{- if .Values.webhook.certificate.generate }}
The webhook requires a TLS certificate. You need to:

1. Generate self-signed certificate:
   kubectl create secret tls {{ include "k8s-chaos.webhookCertSecretName" . }} \
     --cert=path/to/tls.crt \
     --key=path/to/tls.key \
     -n {{ .Release.Namespace }}

2. Update the ValidatingWebhookConfiguration with the CA bundle

Or use cert-manager (recommended):
   helm upgrade {{ .Release.Name }} k8s-chaos/k8s-chaos \
     --set webhook.certificate.certManager=true \
     --set webhook.certificate.generate=false

{{- else if .Values.webhook.certificate.certManager }}
Webhook certificate will be managed by cert-manager.
Make sure cert-manager is installed in your cluster.

Check certificate status:
   kubectl get certificate -n {{ .Release.Namespace }}
{{- end }}
{{- end }}
